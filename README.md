Downloading:

Do a "git clone --recursive https://github.com/grenaud/schmutzi.git"

Installation:

1) Build Bamtools first:

    cd bamtools/   
    mkdir build/   
    cd build/
    cmake ..
    make 
    cd ../..

2) type "make"


Programs:


mitochondrialDeam	Consensus calling for mitochondrial data
mtCont  	   	Contamination estimate for mitochondrial data
